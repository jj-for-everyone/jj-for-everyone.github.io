# Terminal basics

The content of this chapter doesn't _really_ belong here.
Understanding the basics of the terminal is unrelated to Jujutsu or version control.
If you're already comfortable in the terminal, you can confidently skip this chapter.

That being said, version control is the first point of contact with the terminal for many people.
If you jump into the book without this knowledge, you might have a bad time.
If you're not 100% confident that you know all the "terminal basics", keep reading.
This stuff is even more generally useful than what you'll learn about Jujutsu later.

This chapter is by no means exhaustive.
There's much more to learn about the terminal.
My intent is to provide you with just enough knowledge for you to fully benefit from the rest of this book.

- CWD
- PATH
- EDITOR
- Comments `#`
- copy-paste with Ctrl+Shift+{C,V}
- copy button on code blocks
